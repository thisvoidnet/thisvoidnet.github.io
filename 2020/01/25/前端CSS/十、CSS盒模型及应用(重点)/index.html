11.文字盒子居中图片和背景区别

  1. 背景图片更改大小 background-size
  2. 背景图片更改位置 background-position
  3. 产品图片一般用img做
  4. 小细节地方用background做

  1. 一般写法

      margin:0;
      padding:0;

  2. 生产写法

    margin:0;

    }
    div,dl,dt,dd,ul,ol,li,h1,h2,h3,h4,h5,h6,pre,form,fieldset,input,textarea,blockquote,p{

    margin:0;

    }

13.行内元素上下内外边距的问题

  1. 行内元素只有左右边距，没有上下边距
  2. 尽量不要给行内元素指定上下的内外边距

14.外边距合并

  1. 同级元素尽量避免外边距合并
  2. 嵌套元素
    1. 给容器加边框
    2. overflow:hidden;

15.盒子的计算尺寸

  1. 外和尺寸计算（元素空间尺寸）
    Element 空间高度 = content height + padding +border + margin
    Element 空间宽度 = content width + padding +border + margin
  2. 内盒尺寸计算（元素大小）
    Element Height = content height + padding +border
    Element 空间宽度 = content width + padding +border

16.学会计算盒子尺寸（重点）

17.padding不影响盒子大小的情况

  1. height 没给定，padding-top、padding-bottom不影响盒子大小
  2. width 没给定，padding-left、padding-right不影响盒子大小

18.搜索趣图

  1. 容器大小，容器上边框
  2. 第一个子元素div对齐问题，margin造成元素位移，padding可以避免位移
  3. 第二个子元素img，margin、padding都可以实现
  4. 第三个子元素ul列表，list-style：none； 颜色、下划线、向右偏移。

19.盒子模型布局稳定性

21.盒子阴影

box-shadow:水平位置 垂直位置 模糊距离 阴影尺寸 阴影颜色 内/外阴影;

值 描述
h-shadow 必须。水平阴影的位置。允许负值。
v-shadow 必须，垂直阴影的位置。允许负值。
blur 可选。模糊距离
spread 可选。阴影的尺寸，影子的大小
color 可选。阴影的颜色
inset 可选，将外部阴影（outset）改为内部阴影
