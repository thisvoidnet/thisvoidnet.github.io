  3. 后代选择器

    1. 后代选择器所有符合的后代都变样式

  1. 子元素选择器

    .nav > li

    1. 加了右括号的选择器为子元素符合，子元素的后代不再继承样式

  2. 课堂练习-测试选择器

  1. 属性选择器一

    /* 获取到拥有该属性的元素*/

    选择器 示例 含义
    E[attr] 存在attr属性即可
    E[attr=val] 属性值完全等于val
    E[attr*=val] 属性值包含val字符并且在任意位置
    E[attr^=val] 属性值包含val字符并且在开始位置
    E[attr$=val] 属性值包含val字符并且在结束位置

  2. 伪元素选择器一

    1. CSS专有

      选择器 示例 含义
      E::first-letter 文本中第一个单词或字
      E::first-line 文本第一行
      E::selection 改变选中文本的样式
      E::before 在E元素内部的开始位置创建一个元素，该元素为行内元素，而且必须要结合content属性使用
      E::after 在E元素内部的结束位置创建一个元素，该元素为行内元素，而且必须要结合content属性使用

  4. CSS书写规范

    1. 空格规范

      1. 【强制】选择器与 { 之间必须包含空格
      2. 【强制】属性名与之后的:之间不允许包含空格，:与属性值之间必须包含空格

    2. 选择器规范

      1. 【强制】当一个rule包含多个selector时，每个选择器声明必须独占一行

      2. 【强制】选择器的嵌套层级应不大于3级。位置靠后的限定条件应尽可能精确

    3. 属性规范

      1. 【强制】属性定义必须另起一行
      1. 【强制】属性定义必须一分号结尾
