    1. 边偏移

    2. 定位模式
      • 静态定位 static
        • 边偏移不适用
      • 相对定位 relative
        • 不脱离文档流，继续占位置
        • 以左上角为基点
      • 绝对定位 absolute
        • 脱离文档流，不占位置
        • 如父级元素没有定位，则以浏览器为基准点
        • 如父级有定位，则以最近的父级为基准点
        • 常用，子元素绝对定位，父元素相对定位，子绝父相
      • 固定定位 fixed
